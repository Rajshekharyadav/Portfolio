# Raj Shekhar Yadav Portfolio

Premium static portfolio website for Raj Shekhar Yadav, built with HTML, CSS, vanilla JavaScript, Three.js, and GSAP.

## Structure

```text
portfolio/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
├── assets/
│   ├── images/
│   ├── icons/
│   └── resume/
└── README.md
```

## Run Locally

Open `index.html` directly in a browser, or serve the folder with any static server.

## Before Launch

- Replace placeholder social links with real LinkedIn, GitHub, and Instagram URLs.
- Add the real resume PDF at `assets/resume/Raj_Shekhar_Yadav_Resume.pdf`.
- Replace project `#` links with real GitHub repositories and live demos.
- Replace temporary Unsplash images with final personal/project screenshots before production launch.
- CDN access is required for Three.js and GSAP animations.

## Deployment

This site is ready for GitHub Pages, Netlify, Vercel, or any static hosting provider.
